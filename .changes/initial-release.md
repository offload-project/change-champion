Initial release of Change Champion - a changesets-inspired tool for PHP/Composer packages.

- `cc init` - Initialize .changes directory with config
- `cc add` - Interactive changeset creation
- `cc status` - Show pending changesets and next version
- `cc version` - Apply changesets and update CHANGELOG
- `cc publish` - Create git tags
- GitHub Actions workflow scaffolding with `--with-github-actions`
